# Grammlin Extension

A browser extension for Swedish language learners that provides grammatical breakdowns and word definitions for selected text.

## Tech Stack

- [WXT](https://wxt.dev/)
- Svelte 5
- Tailwind CSS + DaisyUI
- TypeScript

## Local Development

```bash
npm install
```

```bash
npm run dev
```

## Build

To build the extension zip, for chrome:
```bash
npm run zip
```

or for firefox:
```bash
npm run zip:firefox
```
