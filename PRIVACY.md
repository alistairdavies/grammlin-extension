# Privacy Policy — Grammlin Web Extension

**Last updated:** 22 March 2026

## What Grammlin does

Grammlin is a browser extension that helps Swedish language learners by analysing highlighted text on web pages. When you select text, it is sent to a server for grammatical analysis, and the results are displayed in a browser side panel.

## Data collected

Grammlin collects **no personal information**. It does not require you to create an account, log in, or provide any identifying details.

The only data used is the **text you highlight** on a web page when the extension has been enabled. This text is sent to the Grammlin analysis server solely for the purpose of providing grammatical breakdowns, definitions, and translations.

## How your data is handled

- Highlighted text is sent to the Grammlin server over HTTPS for analysis.
- The server processes the text and returns results, the text is not handled in any other way.
- The server that receives the text to analyse has request logging, but does not log or store the text content that has been sent in the request.
- No analytics, tracking, or advertising services are used.
- No data is shared with or sold to third parties.

## Extension Permissions

- **Active Tab**: Grants temporary access to the current tab only when you click the extension icon. The extension does not have access to any other tabs or pages.
- **Scripting**: Used to inject a content script into the active tab to detect text you highlight. The extension only reads text you actively select — it does not read or access any other page content.
- **Side Panel**: Used to display analysis results.

## Changes to this policy

Any changes to this privacy policy will be reflected in this document with an updated date.
